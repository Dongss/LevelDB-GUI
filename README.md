# LevelDB-GUI

A [LevelDB] GUI, built with [Electron](https://github.com/electron/electron) and [levelup](https://github.com/Level/levelup)

## Notes

No release yet !

## Requirements

NodeJs >= 6.0.0

## Development

`npm install`

`npm run dev`

`LevelDOWN` rebuild: `npm run electron-rebuild`

## Status

Work in progress. Contributions welcome.

We are working on branch `dev` now.
